Roadmap

Découvrez notre vision et les fonctionnalités prévues pour Spring-Fullstack-Speed

Cette roadmap présente les plans de développement futurs pour Spring-Fullstack-Speed. Elle est susceptible d'évoluer en fonction des retours de la communauté et des priorités du projet.

Version actuelle

Version 1.0.1

Juin 2025

Version stable actuelle avec les fonctionnalités de base.

  • Génération d'applications Spring Boot complètes
  • Support pour React, Vue.js, Angular, Thymeleaf et JTE
  • Génération d'entités, DTOs et opérations CRUD
  • Authentification JWT et OAuth2
  • Support pour MySQL, PostgreSQL, MongoDB et H2
  • Configuration Docker et Kubernetes
  • Tests automatisés
Prochaine version

Version 1.1.0

T3 2025

Amélioration des fonctionnalités existantes et ajout de nouvelles fonctionnalités.

Nouvelles fonctionnalités

  • Support GraphQL complet
  • Système d'extensions amélioré
  • Thèmes pour le frontend
  • Support WebSockets avancé

Améliorations

  • Optimisation des templates React et Vue.js
  • Mise à jour vers les dernières versions de Spring Boot et Angular
  • Support amélioré pour les microservices
  • Génération de documentation API OpenAPI 3.1

Corrections

  • Correction des problèmes connus avec la génération des tests
  • Amélioration de la compatibilité avec différentes versions de Node.js
  • Résolution des problèmes de génération sur Windows
Moyen terme

Version 1.2.0

T4 2025

Expansion majeure des fonctionnalités pour supporter des cas d'usage plus avancés.

Nouvelles fonctionnalités

  • Frontend SSR/SSG avec Next.js et Nuxt.js
  • Mode Microservices pour architectures distribuées
  • AI Assistant intégré pour aider à la génération
  • Marketplace de templates
  • Intégration CI/CD avancée

Améliorations

  • Interface utilisateur de ligne de commande enrichie
  • Optimisations de performance pour grands projets
  • Outils de migration entre versions
  • Tests de performance natifs avec JMH
Long terme

Version 2.0.0

2026

Version majeure avec des changements importants et des fonctionnalités innovantes.

Nouvelles fonctionnalités

  • Interface graphique web pour configurer et générer des projets
  • Cloud Generation avec prévisualisation instantanée
  • Multi-architecture (hexagonale, Clean Architecture, DDD)
  • IA pour la génération de code métier personnalisé
  • Support multiplateforme (mobile, desktop)

Améliorations

  • Support natif pour Java 21+
  • Intégration avec plus de services cloud
  • Métriques et analytics avancés
  • Génération incrémentale pour performances optimales

Initiatives continues

Documentation

  • Mise à jour continue de la documentation
  • Création de tutoriels vidéo
  • Publication d'articles et guides

Maintenance

  • Support des dernières versions de Spring Boot
  • Mise à jour des dépendances
  • Corrections de bugs

Communauté

  • Support des contributeurs
  • Programmes de mentorat
  • Événements et webinaires

Votre avis compte

Cette roadmap n'est pas figée, elle évolue en fonction des retours de la communauté.
