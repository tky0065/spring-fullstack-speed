Commandes SFS v1.0.2

Guide complet de toutes les commandes disponibles

sfs app

Génération Principal

Génère une nouvelle application Spring Boot fullstack complète.

Exemples

sfs app
sfs app --name=my-app --package=com.example --db=postgresql --frontend=react

Options

Option Description Valeurs
--name Nom de l'application String
--package Package Java de base String
--db Base de données h2, mysql, postgresql, mongodb
--frontend Framework frontend none, react, vue, angular, thymeleaf, jte
--auth Type d'authentification none, jwt, oauth2, session

sfs entity

Génération Alias: e

Génère une nouvelle entité Java avec son repository.

Exemples

sfs entity --name=Product
sfs entity --name=Product --fields=name:string,price:double,description:string[required]

Options

Option Description Valeurs
--name Nom de l'entité String
--fields Champs de l'entité Liste de nom:type[validations]
--relations Relations avec d'autres entités Liste de type:entité:field
--package Package de l'entité String

sfs dtos

Génération Alias: d

Génère des DTOs pour une entité existante.

Exemples

sfs dtos --entity=Product
sfs dtos --entity=Product --types=basic,create,update --mapping=mapstruct

Options

Option Description Valeurs
--entity Nom de l'entité String
--types Types de DTOs à générer basic, create, update, view, all
--mapping Outil de mapping entité-DTO manual, mapstruct, modelmapper

sfs crud

Génération Alias: c

Génère les opérations CRUD pour une entité existante.

Exemples

sfs crud --entity=Product
sfs crud --entity=Product --pagination=true --sorting=true --frontend=true

Options

Option Description Valeurs
--entity Nom de l'entité String
--pagination Activer la pagination true, false
--sorting Activer le tri true, false
--frontend Générer des composants frontend true, false

sfs serve

Développement Alias: s

Démarre un serveur de développement pour votre application.

Exemples

sfs serve
sfs serve --backend --frontend --port=8080

Options

Option Description Valeurs
--backend Démarrer le serveur backend true, false
--frontend Démarrer le serveur frontend true, false
--port Port du serveur backend Numéro de port

sfs test

Développement Alias: t

Exécute différents types de tests sur votre application.

Exemples

sfs test
sfs test --type=unit

Options

Option Description Valeurs
--type Type de tests à exécuter unit, integration, e2e, all
--coverage Générer un rapport de couverture true, false
--watch Exécuter en mode watch true, false

sfs docker

Déploiement

Gère les conteneurs Docker pour votre application.

Exemples

sfs docker build
sfs docker compose up

Sous-commandes

Commande Description
build Construit une image Docker
compose Gère les configurations docker-compose
push Pousse une image vers un registry
run Exécute l'application dans un conteneur

sfs kubernetes

Déploiement Alias: k8s

Gère les déploiements Kubernetes pour votre application.

Exemples

sfs kubernetes apply
sfs kubernetes apply --env=dev

Sous-commandes

Commande Description
apply Applique les configurations K8s
generate Génère de nouvelles configurations K8s
delete Supprime les ressources K8s
status Affiche l'état du déploiement

sfs doctor

Utilitaire

Analyse votre projet pour détecter et corriger les problèmes éventuels.

Exemples

sfs doctor
sfs doctor --fix

Options

Option Description Valeurs
--fix Tenter de corriger les problèmes true, false
--type Type de problèmes à chercher structure, dependencies, config, all

sfs upgrade

Utilitaire

Met à jour votre projet vers une version plus récente de Spring-Fullstack-Speed.

Exemples

sfs upgrade
sfs upgrade --target=1.1.0

Options

Option Description Valeurs
--target Version cible numéro de version
--backup Créer une sauvegarde avant la mise à jour true, false

sfs payment NOUVEAU

Business v1.0.2

Génère un système de paiement complet avec intégration de passerelles de paiement (Stripe, PayPal, etc.).

Exemples

sfs payment
sfs payment --provider=stripe,paypal --subscription=true --invoicing=true --taxes=true
sfs payment --provider=stripe --mobile-pay=true --international=true

Options

Option Description Valeurs
--provider Fournisseurs de paiement à intégrer stripe, paypal, braintree, adyen, mollie
--subscription Activer la gestion des abonnements true, false
--invoicing Activer le système de facturation true, false
--taxes Configurer la gestion des taxes true, false
--refunds Implémenter la gestion des remboursements true, false
--reporting Générer des rapports financiers true, false
--mobile-pay Activer le paiement mobile (Apple Pay, Google Pay) true, false
--international Support des devises internationales true, false
--fraud-detection Ajouter la détection de fraude true, false
